Manual Install

  1. Install Docker and VirtualBox

    The first step is to install Docker and VirtualBox on your system. Docker VirtualBox

  2. Add VirtualBox to PATH Environment Variables

    VirtualBox must be added to the Local Machines PATH environment variables. To do this open up the Registry Editor and navigate to the path Computer\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Environment Right click and select modify and append ';C:\Program Files\Oracle\VirtualBox' to the end of the string. A restart will be needed to have this change take effect. This is done so that VBoxManage can be used to automate start up of VMs.

  3. Enable Windows Network Discovery

    In control panel navigate to 'Control Panel\Network and Internet\Network and Sharing Centre\Advanced sharing settings' From here enable 'Turn on network discovery'. This has to be enabled to allow VMs to communicate with the Docker containers.

  4. Restart PC

    Restart your PC for the above changes to take affect.

  5. Setup Virtual Networks

    In order to setup the correct virtual networks the setup_networks.ps1 script has to be run. This is located in the source directory of the downloaded application. You will be prompted after restarting the application to install.
